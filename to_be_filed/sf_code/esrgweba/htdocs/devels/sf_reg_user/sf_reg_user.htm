How To Become A SourceForge Registered User

Before submitting bug reports, suggestions, and feature requests, or requesting support, or collaborating with us, we require you to become a SourceForge registered user.  We hope that this is a minimal inconvenience.  The SourceForge new user registration process begins at this web page and is straightforward.

Note that if you simply download and use any of the materials provided, we do not require you to register.  You only must register if you wish to report a bug, make a suggestion, request a new feature, request support, or collaborate.

In a nutshell, the reason we require registration is that the registration is required to take full advantage of the resources provided by SourceForge.  The detailed reasons are:

  • You must register to use to use the bug tracker, feature request tracker, or support tracker at SourceForge.  These database systems are the most convenient way for us to process the information, requests, or suggestions you might submit.
  • Registration gives us a simple and reliable way to contact you (through your SourceForge mail alias).  Without registration, we would need to collect personal information, which is unwieldly.
  • If you desire to collaborate with us, registration is required so that you can make CVS commits and perform other collaborative tasks.
