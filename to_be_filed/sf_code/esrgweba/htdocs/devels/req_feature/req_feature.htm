How To Request A New Feature Or Enhancement

We welcome and are grateful for all feature and enhancement requests.

The only way that you can request a feature or enhancement is to use SourceForge's feature request tracking system.  Here are the steps you should follow:

  • You must be a registered SourceForge user.  (If you are not, instructions are here.)
  • Please review the information about how categories and groups are used in the feature request tracking system.  This will help you to complete the category and group fields when you submit your feature or enhancement request.
  • Submit your feature request using SourceForge's feature request tracking system for The ESRG Tool Set.

Thanks a lot for your help!
