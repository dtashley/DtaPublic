Policies, Procedures, Checklists, And Other Information For Developers And Collaborators

This page links to information useful for:

  • Developers:  those working on the tool sets or the book.
  • Collaborators:  those working with us on research endeavors.
  • Individuals who wish to rebuild the tool sets or the book:  usually for the following reasons:
    • To enhance or customize the tool sets or the book for a specific application.
    • To rebuild the tool sets so that debugging modifications can be made in an effort to find and correct a bug.
