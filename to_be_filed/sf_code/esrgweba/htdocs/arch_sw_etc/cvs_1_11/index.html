CVS 1.11 Windows Executable Download

This .ZIP file contains CVS 1.11 for Windows.  To install, the executable only needs to be placed in a directory which is always on the path.  For Windows 2000®, this is usually in the directory C:\WINNT\SYSTEM32.  For other versions of Windows®, search for utilities such as REGEDIT, PING, etc., and place the CVS executable in the same directory.

To test the installation, open a DOS shell and type "cvs".  If a message from the CVS program is displayed, installation is successful.
