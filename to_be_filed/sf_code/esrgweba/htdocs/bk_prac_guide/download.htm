Download ESRG Microcontroller Software Book ("A Practitioner's Guide ...")

The most up-to-date version of the book (the tip of CVS archives) is packaged as a .PDF file each time The ESRG Tool Set is released.  To download this .PDF file:

  • Go to the file download page.
  • Locate the release file with a name of the form esrg_ubka_<v_n>.pdf, where <v_n> is the most recent version number.  (For background, it may also be helpful to refer to the version numbering policy and/or the file release policy.)
  • Right-click on the download link (the exact procedure varies depending on the browser being used), and save the file to a suitable location on your Windows® computer.
