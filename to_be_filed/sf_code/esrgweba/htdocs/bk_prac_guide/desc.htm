ESRG Microcontroller Software Book ("A Practitioner's Guide ...") Description

The ESRG book under construction, tentatively entitled "A Practitioner's Guide To The Design And Development Of Small Microcontroller Software", is a unique undertaking on the Internet.  The unique features are:

  • The book contains many insights and many mathematical results that are (to the best of our knowledge) not available anywhere else.
  • The book is not constrained for length, since at this time it is published only electronically on the Internet.  This means that there is room for unlimited numbers of examples, peripheral observations, conjectures, unsolved problems, etc.
  • The book is collaborative (anyone can pitch in).
  • The book is open-source (however, licensing issues are still being worked out), so that all LaTeX source code and other files are under version control and available in their entirety via anonymous CVS.  This means, for example, that anyone can use this book as an example of how to lay out the LaTeX source files of another book.

The title of the book accurately describes its content.  It is about small microcontroller software.
