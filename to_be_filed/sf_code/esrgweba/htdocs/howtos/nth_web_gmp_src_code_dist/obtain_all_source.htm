How To Obtain All Source Code For The Web-Based Number Theory Utilities Here At This Site

The web-based number theory utilities at this site are powered by the GMP.  It is our understanding that the licensing terms for the GMP do not place any requirements on us (because users only run the program, rather than download it).  However, in the open-source spirit of things:

  1. We have made all technical details of the operation of these web pages on SourceForge available here (using the GMP on SourceForge isn't so easy J).
  2. We have made made all C-language source code available under the GPL (click here to view the C-language source code using ViewCVS for the utility which runs on the server to support the PHP scripts).
  3. We have made all PHP script code available under the GPL (click here to view the PHP scripts using ViewCVS).
