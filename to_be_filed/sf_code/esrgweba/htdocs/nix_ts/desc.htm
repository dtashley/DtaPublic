ESRG *nix Tool Set Description

The ESRG *nix tool set is an eclectic set of non-graphical *nix utilities that perform functions useful in microcontroller work.  Nearly all embedded software development is done on Windows® platforms, and so the Unix utilities are "utilities of convenience"--when convenient, a *nix utility which behaves identically to a Windows® utility is created.

Presently, the ESRG *nix tool set is unpopulated (it does not exist yet).
