ESRG Web Content Description

The entire ESRG web content maintained at SourceForge (including PHP scripts and associated utilities) is under version control and can be downloaded piecemeal or in its entirety by anyone using either anonymous CVS access or the ViewCVS interface to CVS archives.

We have placed the web content under version control for the following reasons:

  • There can be upload and download accidents which would cause us to lose the web content.
  • Users may want to examine the PHP scripts that generate the displayed HTML for certain pages, or rebuild executable utilities that support the PHP scripts.
  • There is no necessary uploading and downloading of files for the esrg.sourceforge.net site--changes are made in CVS and once per week the server automatically updates web content.
