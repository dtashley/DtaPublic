Report An ESRG Windows® Tool Set Bug

The mechanism used to report bugs or suspected bugs is to use the tracking system provided by SourceForge.  Here are the steps to follow:

  • Consult this page to determine the correct category and group to use when entering the bug into the tracking system.
  • Open a new SourceForge tracker item for this project and include as much information as you have about the bug.  Ideally, the information included would be the simplest set of information and the simplest procedure to reproduce the bug.

Thanks a lot for your time in helping us make the tool set better!
