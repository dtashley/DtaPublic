Download ESRG Windows® Tool Set

The ESRG Windows® Tool Set is packaged as a single InstallShield Express installation executable.  To download this executable:

  • Go to the file download page.
  • Locate the release file with a name of the form esrg_win_ts_<v_n>.exe, where <v_n> is the most recent version number.  (For background, it may also be helpful to refer to the version numbering policy and/or the file release policy.)
  • Right-click on the download link (the exact procedure varies depending on the browser being used), and save the file to a suitable location on your Windows® computer.

Once the .exe installation executable has been downloaded, the ESRG Windows® Tool Set can be installed using the following steps: 

  • Using Windows® Explorer, navigate to the file that was downloaded and double-click on it to run it.
  • In nearly all cases, accept all installation defaults.
