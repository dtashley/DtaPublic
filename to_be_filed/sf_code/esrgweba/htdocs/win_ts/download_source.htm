Download ESRG Windows® Tool Set Source Code

All of the source code for the Windows® tools is kept in the CVS module esrgpcpj.  There are two ways to obtain this source code, depending on whether the source code desired is the newest source code or the source code for a specific release.

  • Source Code For A Specific Release
    • Visit the file download page and find the tar.gz file with a name of the form esrg_win_ts_src_<v_n>.tar.gz, where <v_n> is the version number of interest.
    • Download the tar.gz file to a convenient location on a Windows® computer.
    • Use WinZip, 7-Zip, or a similar product to decompress the .tar.gz file, preserving the directory hierarchy.
    • Move the directory esrgpcpj so that it is a top-level directory of the c: drive, i.e. so that c:\esrgpcpj exists. 
  • Newest Source Code
    • The newest source code can be obtained via anonymous CVS access.  This page provides instructions for obtaining all source code via anonymous CVS access.  The only module required is esrgpcpj.

After the source code has been obtained and the directory esrgpcpj has been moved to be c:\esrgpcpj (if necessary), the source code can be built using Microsoft Visual C++.  Instructions are provided at this page.
