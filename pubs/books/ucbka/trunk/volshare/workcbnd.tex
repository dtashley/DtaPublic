%%%% Below the bindings between volumes and chapters are specified.  Cross-references   %%%%
%%%% within a chapter are not a problem, as the chapter is defined as the atomic unit.  %%%%
%%%% However, cross-references between chapters need to be treated with more caution,   %%%%
%%%% as the volumes are compiled separately and chapters may shift around between       %%%%
%%%% volumes.  The constants below centralize these associations so that if they change %%%%
%%%% the modifications need only be done in one place.  Each chapter is assigned a      %%%%
%%%% 4-character key which is unique throughout the work.  This 4-character key is the  %%%%
%%%% basis for all of the names below.  Constants for citing every chapter are defined  %%%%
%%%% below.  In an individual volume, the constants for the chapters contained in that  %%%%
%%%% volume are redefined as the empty string, so that cross-references within the      %%%%
%%%% same volume are shortened.                                                         %%%%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%%% Chapters below should be included in alphabetical order of the 4-character tags.
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%%%
